How To Use This Tool

  1. Open the Pitching Ninja Dropbox in another browser tab
  2. Find a pitcher's grip image and save it to your Downloads folder
  3. Come back here, click PICK IMAGE on the correct slot
  4. Preview will load — then click SAVE to download it with the exact correct filename
  5. Move all downloaded files into the grips/ folder next to index.html
  6. When done, open index.html — all images will auto-load in the pitch design tool
Also supports video files (.mp4 / .mov) for Paul's grip videos. For Paul's pitches, use the PAUL SKENES section below.
